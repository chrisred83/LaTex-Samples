El objetivo de esta plantilla es brindar un punto de partida funcional para crear un documento en LaTex y servir de guía para quien aín no ha utilizado
LaTex. Recordando que esta plantilla no reemplaza la consulta de la documentación y otros ejemplos en línea para aprovechar al máximo su potencial.

En la carpeta Documento se encuentran:

*Carpeta imgs1 para almacenar imágenes.
*PlantillaDoc.tex que es el formato del documento en LaTex incluye comentarios con una breve descripción.
*referencias.bib que es el archivo utilizado para agregar referencias al final del documento.

Estos funcionan como un ejemplo descriptivo de un documento en LaTex que cubre una serie de caracteristicas comúnes para cualquier documento y en caso de
requerirse un cambio este pueda ser modificado de acuerdo a las necesidades de quien lo consulta.
